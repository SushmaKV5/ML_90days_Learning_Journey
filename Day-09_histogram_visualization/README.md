# Day 09 — Histogram Visualization Project

## Objective
Visualize the distribution of numerical features using histograms.

## Description
This project generates histograms for all numerical columns in the dataset to understand how values are distributed.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Concepts
- Data distribution
- Frequency analysis
- Exploratory Data Analysis (EDA)

## How to Run
python histogram_visualization.py

## Learning Outcome
Understanding data distribution helps identify skewness, outliers, and variance before applying Machine Learning models.
