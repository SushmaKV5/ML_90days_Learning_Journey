# Day 05 — Data Cleaning Tool (Missing Values & Duplicates)

## Objective
Build a data cleaning script to handle missing values and remove duplicate records using Pandas.

## Topics Covered
- Handling missing values
- Filling null values using mean
- Removing duplicate rows
- Saving cleaned dataset

## Tools
Python, Pandas

## Workflow
Dataset Generation (Day 03) → Inspection (Day 04) → Cleaning (Day 05)

## Key Steps
1. Loaded dataset using Pandas
2. Checked missing values
3. Filled null values with column mean
4. Removed duplicate records
5. Saved cleaned dataset

## Learnings
Understood the importance of data cleaning in Machine Learning pipelines and how poor data quality can affect model performance.
