# Day 06 — Exploratory Statistics Calculator

## Objective
Build a statistical analysis tool to calculate key descriptive statistics of a dataset.

## Topics Covered
- Mean
- Median
- Standard Deviation
- Variance
- Min & Max values
- Statistical Summary using describe()

## Tools
Python, Pandas

## Workflow
Data Generation (Day 03) → Inspection (Day 04) → Cleaning (Day 05) → Statistical Analysis (Day 06)

## Key Features
- Automated statistical report
- Works on any CSV dataset
- Quick dataset understanding

## Learnings
Learned how to perform Exploratory Data Analysis (EDA) using statistical measures to understand data distribution before applying Machine Learning models.
