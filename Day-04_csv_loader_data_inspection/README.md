# Day 04 — CSV Loader & Data Inspection Tool

## Objective
Build a tool to load and inspect datasets using Pandas for initial data analysis.

## Topics Covered
- Loading CSV files using Pandas
- Data inspection (head, shape, columns)
- Data types analysis
- Statistical summary
- Missing value detection

## Tools
Python, Pandas

## Workflow
Generated dataset (Day 03) → Loaded and inspected dataset (Day 04)

## Learnings
Learned how to explore datasets before preprocessing, which is a crucial first step in any Machine Learning project.
