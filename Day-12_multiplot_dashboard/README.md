# Day 12 — Multi-Plot Visualization Dashboard

## Objective
Create a dashboard that displays multiple visualizations in a single figure.

## Visualizations Included
- Histogram (Age distribution)
- Scatter plot (Experience vs Salary)
- Boxplot (Salary outliers)
- Bar chart (Average salary by experience)

## Tools Used
Python, Pandas, Matplotlib

## Key Concepts
- Data visualization
- Dashboard creation
- Exploratory Data Analysis (EDA)

## How to Run
python multiplot_dashboard.py

## Learning Outcome
Combining multiple plots into a dashboard helps quickly understand different aspects of the dataset in one view, which is useful for exploratory data analysis and reporting.
