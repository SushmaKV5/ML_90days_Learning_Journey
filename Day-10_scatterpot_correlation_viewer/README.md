# Day 10 — Scatter Plot Correlation Viewer

## Objective
Visualize relationships between numerical variables using scatter plots and correlation analysis.

## Features
- Automatically detects numerical columns
- Generates scatter plots for feature pairs
- Calculates correlation coefficient

## Tools Used
Python, Pandas, Matplotlib

## Key Concepts
- Scatter plots
- Feature relationships
- Correlation analysis
- Exploratory Data Analysis (EDA)

## How to Run
python scatter_correlation_viewer.py

## Learning Outcome
Scatter plots help identify relationships between features, which is important for feature selection and Machine Learning model development.
