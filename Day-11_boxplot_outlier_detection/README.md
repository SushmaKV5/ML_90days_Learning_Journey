# Day 11 — Boxplot Outlier Detection

## Objective
Identify outliers in numerical features using boxplots.

## Description
This project generates boxplots for each numerical column to visually detect extreme values (outliers).

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Concepts
- Outlier detection
- Boxplot visualization
- Exploratory Data Analysis (EDA)

## How to Run
python boxplot_outlier_detection.py

## Learning Outcome
Boxplots help detect unusual values in datasets, allowing better preprocessing before applying Machine Learning models.
