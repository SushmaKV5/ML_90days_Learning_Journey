# Day 07 — Data Summary CLI Tool

## Objective
Develop a command-line tool to generate a quick summary report of any CSV dataset.

## Features
- Displays dataset shape
- Shows column names and data types
- Detects missing values
- Generates statistical summary
- Counts unique values per column

## Tools
Python, Pandas

## Workflow
Dataset Generation (Day 03) → Inspection (Day 04) → Cleaning (Day 05) → EDA Statistics (Day 06) → CLI Summary Tool (Day 07)

## How to Run
```bash
python data_summary_cli.py
